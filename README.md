# Bulgarian Broadband Ring

Interactive static web map for a 100Gb Bulgarian broadband/research network ring.

## Route

Sofia -> Blagoevgrad -> Plovdiv -> Sliven -> Shumen -> Varna -> Ruse -> Svishtov -> Veliko Turnovo -> Gabrovo -> Pleven -> Dolna Mitropolia -> Sofia

All links are rendered as 100Gb.

Rozhen is added as a 1Gb spur with a link to Plovdiv.

The dashboard includes a compact English/Bulgarian language switch for the interface, city labels, selected segment details, and academic-node pop-ups.

## Boundary Data

The Bulgaria country outline is based on the public-domain geoBoundaries BGR ADM0 GeoJSON dataset:
https://www.geoboundaries.org/api/current/gbOpen/BGR/ADM0/

## Run

Open `index.html` in a browser. No build step or external dependencies are required.
